# Web-Travel-Chonburi

# รายละเอียดเกี่ยวกับเว็บไซต์แนะนําสถานที่ท่องเที่ยวจังหวัดบ้านเกิด
ประกอบด้วย 3 หน้า 
# หน้าหลัก (Home) 
- แสดงภาพรวมของจังหวัด 
- นําเสนอไฮไลท์สถานที่ท่องเที่ยว 
- จัดวางองค์ประกอบที่น่าสนใจ  
# หน้ารายละเอียดสถานที่ท่องเที่ยว (Detail) 
- แสดงข้อมูลสถานที่ท่องเที่ยว 5 แห่ง
- มีรูปภาพและคําอธิบาย 
# หน้าติดต่อ (Contact)  
- มีฟอร์มสําหรับส่งข้อความ 
- แสดงข้อมูลการติดต่อ
